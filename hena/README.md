# 🛡️ Hena Guardian — AI Life Agent v2

Hena Guardian is a futuristic AI-powered personal life assistant built with pure HTML, CSS, and JavaScript.
It combines emotional intelligence, productivity assistance, reminder management, AI integrations, and personalized recommendations into one sleek cyberpunk-inspired interface.

---

# ✨ Features

## 🧠 Emotional Intelligence Engine

Hena detects user emotions in real time using NLP-style pattern matching.

Supported emotional states:

* Happy
* Sad
* Stressed
* Excited
* Neutral

The assistant dynamically changes:

* Tone
* Response style
* Recommendation system
* EQ mode

---

## 🔔 Smart Reminder System

Users can:

* Add reminders
* Delete reminders
* View reminders instantly
* Store reminders locally in browser memory

Example:

```text
Add reminder: Drink water at 5PM
```

---

## 🌱 Habit Tracking

Hena observes recurring keywords and behaviors to:

* Detect habits
* Suggest routines
* Provide productivity recommendations
* Learn user patterns over time

---

## ✨ Personalized Recommendation Engine

Hena automatically gives recommendations based on:

* Stress level
* Productivity patterns
* Sleep-related messages
* Motivation signals
* Emotional state

Examples:

* Breathing exercises
* Focus strategies
* Sleep tips
* Motivation guidance

---
